EVE Item Diff

Paste in lists of EVE items. They will be diffed, and the missing items in each list will be shown below in a format friendly for importing back into EVE (e.g. via Multibuy). All EVE text formats should be supported and be interchangeable within each input box.

EVE Item Diff uses a custom parser and differ written in Rust to perform its function. The libraries are accessed via WASM.
